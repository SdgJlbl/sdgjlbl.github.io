MLV-tools

MLV-tools is a toolbox to help with versioning and reproducibility of Machine Learning experiments. Its main features include:

  • a command to transform Jupyter Notebooks in parametrable and executable Python scripts, allowing for easy functional testing and keeping a unique source of truth from exploration to production to debugging
  • a layer on Data Version Control, to harnest the best of this powerful tool while removing most of the tediousness and the risk of inconsistencies

You can learn more about the project on GitHub. Check out the tutorials!

MLV-tools is also available on PyPI.
